# Tic-Tac-Toe

## Application Quick-start Commands

* To play game:

  * `cd` into `App` dir

  * run app w/ `dotnet run` 

* To run tests:

  * `cd` into root dir

  * run app w/ `dotnet test` 

## Dependencies

* `moq` - most popular and friendly mocking library for .NET

* `dotnet` - .Net Runtime

## Prerequisites

There are no prerequisites for this project.
